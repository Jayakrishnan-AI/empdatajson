# 📊 Employee Data Analysis using Python

## 📌 Project Overview
This project focuses on analyzing employee data stored in a JSON file using Python.  
The goal is to perform data cleaning, exploration, and visualization to gain meaningful insights.

---

## 📂 Dataset Information
The dataset contains employee details such as:

- Employee ID
- First Name & Last Name
- Email
- Phone Number
- Hire Date
- Salary
- Department ID
- Profile Image

---

## 🛠️ Technologies Used

- Python 🐍
- Pandas 📊
- Matplotlib 📈
- Seaborn 🎨
